# Portfólio · Rodrigo Italiani

Site pessoal onde eu mostro o que construo: automações e sistemas sob medida pra empresa. Atendimento de lead no WhatsApp, integração com CRM, relatório que se monta sozinho, esse tipo de coisa.

**No ar:** https://rodrigoitaliani.github.io/portfolio/

## O que é

Uma página única, em formato bento grid, com os projetos em destaque e o estudo de caso de cada um abrindo ali mesmo, sem sair da página. Feita pra carregar rápido e ir direto ao ponto.

## Stack

HTML, CSS e JavaScript puro, sem framework. Hospedado no GitHub Pages.

## Rodar local

Precisa só de Python:

```
python -m http.server 4599
```

Aí é só abrir http://localhost:4599 no navegador.

## Contato

WhatsApp e e-mail estão no site. Se na sua operação tem uma tarefa que se repete todo dia e come o tempo da equipe, provavelmente dá pra automatizar.
